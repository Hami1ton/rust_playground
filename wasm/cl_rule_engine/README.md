# cl_rule_engine

## Env

- WSL2(Windows11)
- node v20.17.0
- npm 10.8.2
- rust 1.80.1
- http-server 14.1.1

## commands

```
- install wasm-pack
cargo install wasm-pack

- install http-server
npm install -g http-server

- run tests
cargo test

- create wasm
wasm-pack build --target web

- start server
http-server

```
