# sqlx_sample

## sqlx

```
- install sqlx cli
cargo install sqlx-cli

- create db
sqlx database create

- create migration files
sqlx migrate add -r user

- apply migration
sqlx migrate run

- drop tables
sqlx migrate revert

```

## sqlite3

```
- connect
sqlite3 sample.db

- select users
select * from user;

- disconnect
.exit
```
